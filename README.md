# GLAMORA Guntur — AI Beauty Salon Marketplace

A premium beauty-tech marketplace built with **TanStack Start (React 19 + Vite 7)** and **Tailwind CSS v4**. Discover verified salons in Guntur, browse curated services, chat with Glow AI, and book appointments end-to-end.

## Tech Stack

- **Framework**: TanStack Start v1 (React 19, SSR-ready, file-based routing)
- **Build**: Vite 7
- **Styling**: Tailwind CSS v4 (via `src/styles.css`)
- **UI Primitives**: shadcn/ui + Radix
- **Icons**: lucide-react
- **Carousel**: embla-carousel-react
- **State / persistence**: localStorage (auth, saved salons, bookings)

## Prerequisites

- **Node.js 20+** (recommended via [nvm](https://github.com/nvm-sh/nvm))
- **npm 10+** (or `bun` / `pnpm` if you prefer)

## Quick Start

```bash
# 1. Install dependencies
npm install

# 2. Start the dev server
npm run dev

# App runs on http://localhost:8080
```

## Available Scripts

| Command          | Description                                  |
| ---------------- | -------------------------------------------- |
| `npm run dev`    | Start Vite dev server with HMR               |
| `npm run build`  | Production build                             |
| `npm run start`  | Serve the production build                   |
| `npm run lint`   | Run ESLint                                   |

## Project Structure

```
src/
  assets/              Images (hero, salons, per-service photography)
  components/
    site/              Nav, Footer, HeroCarousel, SalonCard, MobileBottomNav
    ui/                shadcn/ui primitives
  lib/
    salons.ts          Salon + service catalog (data layer)
    auth.ts            Client-side auth + bookings (localStorage)
  routes/              File-based routes (TanStack Router)
    __root.tsx         Root layout (html/head/body shell)
    index.tsx          Home (hero carousel, categories, featured salons)
    salons.tsx         Salon listings + search
    salons.$id.tsx     Salon profile
    services.tsx       Service marketplace with category filters
    services.$slug.tsx Service detail page
    book.$id.tsx       Multi-step booking flow
    booking-success.tsx
    glow-ai.tsx        AI assistant chat
    offers.tsx         Promotions
    signin.tsx         Sign in / Google quick-login
    dashboard.tsx      User dashboard
  styles.css           Tailwind v4 + design tokens
  router.tsx           Router config
```

## Routing Notes

This project uses **TanStack Router** (not React Router). Routes are defined
by files under `src/routes/` and `src/routeTree.gen.ts` is auto-generated by
the Vite plugin — never edit it manually.

## Deployment

The project targets edge runtimes (Cloudflare Workers compatible).
Standard build:

```bash
npm run build
```

Output goes to `.output/` and can be deployed to any Node 20+ or edge host.
